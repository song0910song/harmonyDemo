# HarmonyOS Demo - 通讯录项目

这是一个基于 HarmonyOS (ArkTS/ArkUI) 开发的学习示例项目。目前主要实现了一个通讯录应用的基础界面布局。

## 📱 项目简介

本项目旨在练习和展示 HarmonyOS 的应用开发能力，目前包含一个完整的通讯录静态页面实现。

## ✨ 功能特性

- **通讯录列表展示**：实现了联系人列表的静态 UI 布局。
- **界面还原**：包含头部导航栏、列表项等标准界面元素。
- **布局练习**：使用了 ArkUI 的常用布局容器（List, Row, Column 等）。

## 🛠️ 开发环境

- **IDE**: DevEco Studio
- **语言**: ArkTS
- **SDK**: HarmonyOS API (请根据实际使用的 API 版本填写，如 API 9 或 API 10)

## 🚀 快速开始

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/song0910song/harmonyDemo.git
   ```

2. 使用 **DevEco Studio** 打开项目根目录。

3. 等待 Sync 完成后，点击运行按钮在模拟器或真机上预览。
